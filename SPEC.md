# OpenDev IDE - Phase 1 技术规格

## 阶段目标

**MVP (Minimum Viable Product)** - 最简可行产品

## 功能范围

### 1. 编辑器核心
- Monaco Editor 集成
- 基础代码编辑（语法高亮、自动补全）
- 多文件标签管理

### 2. 基础 AI 能力
- 单模型接入（OpenAI API）
- 基础的代码补全/生成
- 对话式 AI 助手面板

## 技术架构

```
┌─────────────────────────────────────────┐
│           UI 层 (React)                 │
│  - 编辑器面板                           │
│  - AI 对话面板                          │
├─────────────────────────────────────────┤
│         核心引擎层                      │
│  - Monaco Editor 包装器                 │
│  - AI Service (OpenAI)                 │
├─────────────────────────────────────────┤
│         Electron 层                     │
│  - 窗口管理                            │
│  - 文件系统访问                        │
└─────────────────────────────────────────┘
```

## 验收标准

- [ ] Monaco Editor 正常加载，可编辑代码
- [ ] 可创建/打开/保存文件
- [ ] AI 对话面板可发送消息并接收回复
- [ ] 应用可打包为可执行文件

## 风险与约束

- 初期仅支持 OpenAI，后续扩展 Claude/Ollama
- 仅支持 Windows（后续跨平台）
- 不包含 VSCode 插件兼容层（Phase 2）
